# Avatar Assets

Put character and enemy avatar images in this folder.

Supported formats are normal browser image formats such as PNG, JPG, JPEG, or WEBP, but the current code expects the PNG file names below.

Do not use Chinese file names.

## Folder

```text
src/main/resources/static/assets/avatars
```

## Required File Names

### Player

- `player_male.png` - 男主角
- `player_female.png` - 女主角

### Female Companions

- `companion_nangongwan.png` - 南宫婉
- `companion_chenqiaoqian.png` - 陈巧倩
- `companion_yuanyao.png` - 元瑶
- `companion_ziling.png` - 紫灵
- `companion_mupeiling.png` - 慕沛灵

### Male Companions

- `companion_lifeiyu.png` - 厉飞雨
- `companion_qiyunxiao.png` - 齐云霄
- `companion_mojiao.png` - 墨蛟
- `companion_xiangzhili.png` - 向之礼
- `companion_fengxi.png` - 风希

### Normal Enemies

- `enemy_bandit.png` - 黑风寨匪徒
