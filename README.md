# 凡人修仙外传

网页端单机修仙回合制 RPG。当前版本为 v0.2，聚焦战斗操作、秘籍学习、法宝使用、道友出战调整和五行试炼扩展。

## 技术栈

- Java 17
- Spring Boot 4.0.5
- Vue 3 CDN 单页前端
- 本地 JSON 自动存档
- Maven

> 当前版本暂未接入 MySQL，存档默认写入 `data/saves.json`。`data/` 已加入 `.gitignore`，不会上传个人存档。

## 运行方式

```powershell
mvn spring-boot:run
```

启动后打开：

```text
http://localhost:8080/
```

也可以先打包：

```powershell
mvn -DskipTests package
java -jar target/demo-0.0.1-SNAPSHOT.jar
```

## 当前功能

### 游戏流程

- 主菜单
- 新游戏角色创建
- 黑底白字全屏故事背景
- 自动存档
- 读取进度
- 游戏主界面

### 角色系统

- 角色名称、性别、五行属性
- 初始境界：炼气期 1 级
- 基础属性：体力、灵力、武力、法力、防御、敏捷
- 经验条与升级
- 筑基期突破
- 初始单人开局
- 地图探索战后事件可结识道友
- 道友出战阵容调整
- 道友好感度属性加成：每 5 点好感度为该道友全属性 +1

### 战斗系统

- 回合制战斗
- 按敏捷决定行动顺序
- 地图探索与野外战斗合并为一个入口
- 战斗胜利后触发探索事件
- 玩家可选择：
  - 普通攻击
  - 技能
  - 丹药
  - 逃跑
- 技能/丹药采用二级菜单选择
- 攻击技能可指定敌人
- 治疗技能/丹药可指定我方队友
- 敌方自动行动
- 五行克制伤害加成
- 状态效果：中毒、灼烧、减速等
- 战斗中可使用已装备法宝
- 每名角色每场战斗只能使用 1 次法宝

### 装备系统

- 装备位：
  - 武器
  - 法器
  - 防具
  - 足具
- 装备属性加成
- 背包装备展示
- 点击装备后选择穿戴角色
- 支持替换装备
- 支持卸下装备

### 法宝系统

- 每名角色可装备 1 件法宝
- 当前法宝：
  - 定身符：眩晕敌人
  - 回春符：恢复队友体力
  - 聚灵符：恢复队友灵力
- 法宝可在坊市购买
- 法宝可在战斗中指定目标使用

### 道具系统

- 丹药恢复体力/灵力
- 突破丹用于境界突破
- 商品和道具显示详细说明
- 秘籍可在坊市购买或五行洞掉落
- 秘籍可让对应五行角色学习新技能

### 地图与副本

- 太南山探索
- 锁妖塔挑战，当前开放到 30 层
- 五行试炼洞：
  - 金行洞
  - 木行洞
  - 水行洞
  - 火行洞
  - 土行洞
- 世界 BOSS、古战场入口预留到后续版本

### 客栈

- 右侧栏顶部入口
- 弹窗确认是否恢复
- 显示出战人数、每人花费、总花费、当前灵石
- 恢复全部体力与灵力

### 图鉴

图鉴已按分类展示：

- 装备图鉴
- 道具图鉴
- 功法图鉴
- 道友图鉴
- 怪物图鉴
- 进度统计

## 主要目录

```text
src/main/java/com/example/demo/game
```

游戏控制器、服务层、数值目录。

```text
src/main/java/com/example/demo/model
```

角色、存档、战斗、装备、道具等数据模型。

```text
src/main/resources/static/index.html
```

Vue 单页游戏前端。

```text
src/main/resources/application.properties
```

应用配置，默认端口 `8080`。
